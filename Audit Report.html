03 · This iteration

Everything shipped in v3

11 new fixes
Performancepage weight + paint
  • CSS minified from 53 KB → 44 KB (−18%). Comments stripped, whitespace collapsed, tokens tightened. New file: styles-bundle.min.css.
  • Instant-paint splash with inline-SVG logo + critical inline CSS. User sees the brand < 200 ms after navigation; React mount fades it out via requestAnimationFrame.
  • Async Google Fonts via the media="print" onload swap pattern, with <noscript> fallback. Fonts no longer block render.
  • Above-the-fold critical CSS inlined in <head>. Splash + body shell rendered before any external CSS arrives.
SEO & sharingsocial cards + indexing
  • Raster OG image: generated a 1200×630 JPG (57 KB) with on-brand gradient, B-mark, headline, service line. Replaces the 2.2 KB SVG that LinkedIn/Slack didn't render reliably.

Privacy & legalcompliance content
  • Privacy policy shipped at /privacy.html — covers data collection, retention windows, third-party processors (Google Fonts, form provider), user rights, cookies (none), security posture.
  • Footer link to privacy + contact email added to the bottom strip of every page.
Build pipelineready to run

  • vite.config.js — single-bundle output, esbuild minification, ES2020 target, CSS code-split disabled (one CSS file).
